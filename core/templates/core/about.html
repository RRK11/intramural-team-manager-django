About This Project

The Intramural Team Manager is a Django-based CRUD application built from a previous Streamlit project. It uses the same PostgreSQL database and existing tables to manage players, teams, roster assignments, and game data in a more professional web framework.

This version improves on the original project by adding authentication, Django Admin, navigation, and a more polished homepage interface.
